## Multi-point Liquid Temperature Logger

A 3‑point temperature data acquisition system I built on Arduino Uno 
using MAX31856 thermocouple amplifiers. It calculates real‑time cooling rates, and shows live data on an OLED display.

### Hardware
- Arduino Uno R3
- 3× Adafruit MAX31856 + K‑type thermocouples
- SSD1309 OLED
